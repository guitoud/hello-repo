Installation
-------------

Pour executer ce script, veuillez tout d'abord créer une base de données MYSQL sur votre serveur web.
Une fois la base créer, vous pouvez peupler votre base à l'aide du script "equetes.sql" situé dans le répertoire sql.

Maintenant vous pouver ouvrir le fichier mysql.php situé dans le répertoire conf et modifier les paramètres de connexion mysql
avec vos paramètres ( nom du servuer, nom de la base de données, login et mot de passe).

Lancer l'application en ouvrant avec l'url http://localhost/calendrier.php ( à modifier en fonction de votre environnement).
